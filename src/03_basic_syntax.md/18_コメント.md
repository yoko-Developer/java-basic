# 03-18.コメント

```
public class CommentExample {

    public static void main(String[] args) {
        int a = 5; // aに5を代入

        /*
         変数bは、aの値を2倍にして格納します。
         ここでbの値を計算してから、結果を表示します。
         */
        int b = a * 2;

        // 結果を表示
        System.out.println("aの値: " + a);
        System.out.println("bの値: " + b);

        /**
         * このメソッドは、引数numを2倍にして返します。
         *
         * @param num 倍にする数値
         * @return numの2倍
         */
        int result = doubleValue(a);
        System.out.println("aの2倍: " + result);
    }

    public static int doubleValue(int num) {
        return num * 2;
    }
}
```
